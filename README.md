# FasterFollow (Next.js Version)

هذا المشروع تم تحويله من Vite React إلى Next.js App Router. 

## الأوامر الأساسية:

- **لتشغيل خادم التطوير (Development):**
  ```bash
  npm run dev
  ```

- **لعمل بناء نهائي للإنتاج (Production Build):**
  ```bash
  npm run build
  ```

- **لتشغيل نسخة الإنتاج (Production Start):**
  ```bash
  npm start
  ```
